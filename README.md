🖥️ Windows Server 2025 Installation Lab
A complete lab documenting the installation and configuration of Windows Server 2025 Standard Evaluation (Desktop Experience) in VirtualBox.

📌 Overview
This lab walks through the full installation of Windows Server 2025 inside Oracle VirtualBox, including VM setup, ISO boot, installation steps, and post‑install configuration.
It is designed as a clean, beginner‑friendly reference for IT students and system administrators.

🧰 Requirements
Oracle VirtualBox

Windows Server 2025 ISO

Minimum VM specs:

4 GB RAM

2 CPUs

50 GB storage

🛠️ Virtual Machine Setup
Create a new VM in VirtualBox

Select Windows Server 2025 (64‑bit)

Assign memory and CPU

Create a virtual hard disk

Attach the Windows Server 2025 ISO

Start the VM

🪟 Windows Server Installation Steps
Select Language, Time, and Keyboard

Click Install Now

Choose Windows Server 2025 Standard Evaluation (Desktop Experience)

Accept license terms

Choose Custom Installation

Select the virtual disk

Begin installation

Set the Administrator password

Complete setup and sign in
